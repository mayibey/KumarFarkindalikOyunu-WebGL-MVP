KUMAR FARKINDALIK OYUNU – HAZIR UNITY PROJESİ

1) Bu klasörü ZIP'ten çıkarın.
2) Unity Hub → Open → bu klasörü seçin.
3) Açılınca yeni bir sahne oluşturun (Oyun).
4) Hierarchy → Create Empty → adı: OyunYoneticisi
5) Inspector → Add Component → OyunYoneticisi
6) Play ▶️

Not: İlk 5 tur kazandırır, sonra kaybettirir. Amaç farkındalıktır.
